I know I'm not the only one who likes keeping a local archive of the best finds on /hr/. To make it easier, I built HighResVault.

It lets you preview threads and 'triage' them in seconds. Just click YES to start an instant download, and the app will 
automatically check back every 10 minutes for new posts in those threads. It’s a set-it-and-forget-it way to keep your folders updated without 
wasting bandwidth.

Intuitive to use, easy to stop. Enjoy the high-res hunt, Anon.

I suggest using the releases version rather than the .py version for easier usage, but if you really wanna use the .py one just use this:

pip install -r requirements.txt
