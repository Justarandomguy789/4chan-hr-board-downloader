I know I'm not the only one who likes keeping a local archive of the best finds on /hr/. To make it easier, I built HighResVault.

It lets you preview threads and 'triage' them in seconds. Just click YES to start an instant download, and the app will 
automatically check back every 10 minutes for new posts in those threads (and by clicking tray, you can place tha application in background and it will automatically download new stuff without you realising). It’s a set-it-and-forget-it way to keep your folders updated without 
wasting bandwidth.

Intuitive to use, easy to stop. Enjoy the high-res hunt, Anon.

I suggest using the .exe version that you find on releases rather than the .py version for easier usage, but if you really wanna use the .py one install everything next to each other and don't forget to do this to make it run:

pip install -r requirements.txt
